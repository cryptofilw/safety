------------------------- КАК ИСПОЛЬЗОВАТЬ СОФТ + ПРАВИЛА -------------------------

Чтобы начать шифрование, вам понадобится ТОЛЬКО ФАЙЛ "start.py" (и программа, которая может запустить этот файл)!!!

В терминал после открытия файла нужно ввести команду:
pip install -r requirements.txt

Дальше следуйте правилам ниже:

0. Не изменять ничего в файлах кода, чтобы ничего не навредить

1. Код поддерживает шифрование только текстовых файлов (формат ".txt")

2. Запустите файл "start.py", с помощью зелёной кнопки запуска
или горячими клавишами: CTRL+R (PyCharm) (важно: убедись, что именно этот файл выбран для запуска)

3. Выберите одну из предложенных функций после запуска, написав
в терминал цифру: 1 - Создать файл, 2 - Зашифровать файл, 3 - Расшифровать файл

4. Следуйте инструкциям из терминала, чтобы избежать ошибок (например вводите числа, а не другие символы)

5. Чтобы взаимодействовать с файлами (".txt"),
их обязательно нужно помещать в папку "safety", иначе программа их не видит!!!

6. Не создавайте два файла с одинаковым именем, тк новый файл заменит старый

7. Если вы нашли баг или ошибку, то пишите мне, TG: @cryptofilw

8. Ваши данные можно взломать только в том случае, если злоумышленник нашёл ваш зашифрованный файл и ключ.
Поэтому рекомендую прятать ключи в надёжные места (например на флешку/папку, о которой знаете только вы)

9. Дополнительная защита: для максимальной безопасности, вы можете зашифровать файл с ключом

10. Если вы потеряете ключ, то расшифровать зашифрованный файл будет НЕВОЗМОЖНО
